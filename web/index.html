Tentang Proyek Ini

Proyek ini bertujuan untuk memvisualisasikan solusi numerik dari model matematika menggunakan metode Runge-Kutta. Data dihasilkan melalui skrip Python dan divisualisasikan di web menggunakan Chart.js. Dengan memisahkan visualisasi ke dalam halaman khusus, diharapkan pengguna dapat lebih fokus pada konteks dan hasil dari masing-masing model.

Data dibangkitkan oleh Python lalu divisualisasikan dengan Chart.js. Pilih halaman sesuai model yang ingin dibahas.
